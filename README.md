# My School Website

## Overview
This is a **school website project** created using **HTML, CSS, and JavaScript**.  
The homepage features a **background video**, interactive buttons, smooth scroll animations, and a sticky footer.  

> ⚠️ This project is **still under development**, and more pages and features will be added in the future.

---

## Current Features
- **Responsive Navbar** with links to Home, About Us, Admissions, Academics, Events, Contact, and Login.
- **Search bar** in the navbar.
- **Homepage with background video** and welcome text.
- **Interactive buttons**: "Learn More" and "Admissions".
- **Fade-in animations** for text and cards.
- **Typing effect** for the homepage title.
- **Cards section** with icons and descriptions.
- **Sticky footer** with social media icons.
- **Smooth scroll** when clicking buttons.
- Fully **responsive design** for desktop and mobile.
